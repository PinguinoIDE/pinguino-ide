Name

Flash.write - write a row to internal flash

Description

void Flash.write ( u16 address, u8 *data )
Write a row of 32 bytes (16 words) to internal flash memory

Syntax 

 flash.write( address, *data );
Parameters
address - start address in flash
data - 32 byte buffer
