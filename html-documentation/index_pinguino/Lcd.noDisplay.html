Name

lcd.noDisplay

Description

Turns the LCD display off.

When off, the display data remains in the LCD's DDRAM, but can be displayed instantly by turning the display back on.

Syntax 

void lcd.noDisplay(void);
Parameters
[None]
Returns/Output;
