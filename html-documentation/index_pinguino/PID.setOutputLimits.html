Name

PID.setOutputLimits

Description

void PID.SetOutputLimits( double Min, double Max )
The PID controller is designed to vary its output within a given range. By default this range is 0-255.

Syntax 

 PID.setOutputLimits( min, max );
Parameters
min - Low end of the range. must be < max (double)
max - High end of the range. must be > min (double)
Returns/Output
None

Example

To scale for use with Pinguino PWM library: 

 PID.setOutputLimits( 0 , 1023 )
