Name

keyboard.begin - initialise ps/2 keyboard

Description

void keyboard.begin( void ) - (P8 Only)
Initialise the ps/2 keyboard library
Connections between the keyboard and Pinguino are:
(ps/2) pin 5 ---> (Pinguino) RB.0 (Clock)
(ps/2) pin 1 ---> (Pinguino) RB.1 (Data)
Clock and Data lines require a 10k pull-up resistor.
Keyboard requires +5V on pin 4, with Gnd on pin 3

Syntax 

 keyboard.begin( void )
Parameters
None
Output/Returns
None
