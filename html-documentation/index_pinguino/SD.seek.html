Name

SD.seek

Description

Move read/write pointer, Expand file size

Syntax

FRESULT SD.seek (FIL *fp, dword ofs) 

      /* fp = Pointer to the file object */
     /* ofs = File pointer from top of file - offset */

Example 

FIL Fil; // File object; Global variable
void loop() {
  ...
  rc = SD.seek(&Fil,Fil.fsize); // point just behind the end of file to append the file 
  ...
  rc = SD.seek(&Fil,(dword)len); // point just after len bytes
  ...
}
