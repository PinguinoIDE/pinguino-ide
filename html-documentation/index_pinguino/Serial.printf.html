Name

Serial.printf

Description

Serial.printf sends a formatted string to the current serial port.

It uses printf.c, a new implementation of the stdlib library (stdlib.c).

This new library provides simple and small printf functionality for use in embedded systems.

The formats supported by this implementation are: 'd' 'u' 'b' 'o' 'c' 's' 'x' 'X' and 'f'.

Zero padding and field width are also supported.

Beta 9.5
