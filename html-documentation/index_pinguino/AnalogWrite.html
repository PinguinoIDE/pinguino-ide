Name

analogWrite - Configure a Pulse Width Modulation on a pin

Description

Pins can be used as 10 bits PWM output.
For PIC18F2550 pin 11 and 12 can be both or individually used.
For PIC32 Olimex Board use pin 0 to 2 for outputs on D0, D1, D2
For PIC18F47J53-A, can be used with pin 4, 5, 6, 7, 17, 22 or 23.
For PIC18F4550 , can be used with pin 11, 12

Since
Beta 5

Syntax 

analogWrite( pin, ratio );
pin (type u8)
(11 / 12 for PIC18F2550) (0..2 for PIC32 Olimex Board)
ratio (type u8)
this value can be fixed between 0 and 1023. The high level time is proportional to this value.
Output
none
