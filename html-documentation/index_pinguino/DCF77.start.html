Name

DCF77.start - start receiving DCF77 timecode signal

Description

void DCF77.start( u8 dcfPin )
Start decoding DCF77 timecode signals.
Creates the following RTC structure:
typedef struct
{
u8 seconds;
u8 minutes;
u8 hours;
u8 dayofweek;
u8 dayofmonth;
u8 month;
u8 year;
u8 nosync; // nosync shows minutes
} RTClock
Maintains it via Timer3 interrupt and calibrates timing against timecodes.

Syntax 

 DCF77.start( pin );
Parameters
pin - Pin connected to DCF77 receiver
Returns/Outputs
None
