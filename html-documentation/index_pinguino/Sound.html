Name

Sound

Description

Play notes using pwm.

Since Beta 9.5

Syntax

Sound(u8 pin, u8 note, u16 duration, u8 vol);

  • @param pin: pin number where buzzer is connected (11 or 12)
  • @param note: Note to Play

Romance Notes {DO,DOd,RE,REd,MI,FA,FAd,SOL,SOLd,LA,LAd,SI,DO_}

English Notes {C,Cs,D,Ds,E,F,Fs,G,Gs,A,As,B,C_}

  • @param duration: Duration in ms
  • @param vol: Volume from 0 to 10

Example 

Sound(11, DO, 250, 10);

Play a DO during 250ms, volume is max.

Buzzer must be connected on pin 11 (CCP2).
