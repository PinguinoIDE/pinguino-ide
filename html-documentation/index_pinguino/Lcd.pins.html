Name

lcd.pins

Description

Sets the Pinguino pin numbers used by the LCD display. The LCD display can be controlled using 4 or 8 data lines and this is set by the data (see Parameters below). If 4 data lines are being used then LCD display pins D0 - D3 should be left unconnected. The LCD display RW pin should be conected to ground (OV).

Since Beta X

Syntax

P8
void lcd.pins(u8 rs, u8 enable, u8 d0, u8 d1, u8 d2, u8 d3, u8 d4, u8 d5, u8 d6, u8 d7);
P32
void lcd.pins(unsigned char rs, unsigned char enable, unsigned char d0, unsigned char d1, unsigned char d2, unsigned char d3,  unsigned char d4, unsigned char d5, unsigned char d6, unsigned char d7);
Parameters (8 data lines)
  • rs = the Pinguino pin number connected to the LCD display RS (Register Select) pin
  • enable = the Pinguino pin number connected to the LCD display E (Enable) pin
  • d0 = the Pinguino pin number connected to the LCD display DB0 pin
  • d1 = the Pinguino pin number connected to the LCD display DB1 pin
  • d2 = the Pinguino pin number connected to the LCD display DB2 pin
  • d3 = the Pinguino pin number connected to the LCD display DB3 pin
  • d4 = the Pinguino pin number connected to the LCD display DB4 pin
  • d5 = the Pinguino pin number connected to the LCD display DB5 pin
  • d6 = the Pinguino pin number connected to the LCD display DB6 pin
  • d7 = the Pinguino pin number connected to the LCD display DB7 pin
Parameters (4 data lines)
  • rs = the Pinguino pin number connected to the LCD display RS (Register Select) pin
  • enable = the Pinguino pin number connected to the LCD display E (Enable) pin
  • d0 = the Pinguino pin number connected to the LCD display DB4 pin
  • d1 = the Pinguino pin number connected to the LCD display DB5 pin
  • d2 = the Pinguino pin number connected to the LCD display DB6 pin
  • d3 = the Pinguino pin number connected to the LCD display DB7 pin
  • d4 = 0
  • d5 = 0
  • d6 = 0
  • d7 = 0

Note: in order to only use 4 data lines, d4, d5, d6 and d7 ALL have to be 0 (zero).

Returns/Output;
[None]
