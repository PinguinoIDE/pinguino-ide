Name

PWM.setPercentDutyCycle - Set PWM duty cycle in percent

Description

void PWM.setPercentDutyCycle ( u8 pin, u8 percent )
Set a percentage duty cycle, allowing max 100 PWM steps.

Syntax 

 PWM.setPercentDutyCycle( pin, percent )
Parameters
pin - PWM pin - 5 for CCP1 or 6 for CCP2
percent - from 0 to 99
Returns/Output
None
